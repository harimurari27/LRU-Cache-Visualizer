What is an LRU Cache?

LRU (Least Recently Used) Cache is a data structure that stores a fixed number of elements. When the cache is full, the least recently used item is removed to make space for new data.

How This Project Works

- Set the *cache size* before using the cache.
- Use *Put* to store a key-value pair in the cache.
- Use *Get* to retrieve a value. The retrieved item moves to the front.
- When full, the *Least Recently Used* item is removed.
